Initial `0.1.0` release of the Open Zirndorf package set.

`@openzirndorf/ui`
- Publish the first reusable React component package for Open Zirndorf apps.
- Ship the initial set of branded UI primitives, including buttons, badges, cards, dialogs, sheets, separators, logos, mascots, and shared utilities.
- Include higher-level page and site patterns such as headers, footers, support banners, event cards, feature cards, participation cards, and section headers.

`@openzirndorf/brand`
- Publish the first shared brand asset package for Open Zirndorf.
- Include reusable logo, favicon, and mascot assets with optimized PNG and WebP delivery variants.

This release establishes the first public baseline for consuming the design system through GitHub Packages.
